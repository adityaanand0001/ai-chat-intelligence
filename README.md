## рҹҡҖ Quick Start (Local Development)

This project is a Vercel-optimized monorepo. Both the Next.js frontend and FastAPI backend live inside the `AI-Chat/` directory.

```bash
# 1. Start the Python Backend (terminal 1)
cd AI-Chat/api
python -m venv venv
venv\Scripts\activate      # Windows
# source venv/bin/activate # Mac/Linux
pip install -r requirements.txt uvicorn
python index.py                    # вҶ’ http://localhost:8765

# 2. Start the Next.js Frontend (terminal 2)
cd AI-Chat
npm install
npm run dev                       # вҶ’ http://localhost:3000
```

**Gemini:** paste your key in the Settings modal (gear icon in sidebar), or add `GEMINI_API_KEY=...` to `AI-Chat/api/.env`.

**Supabase:** run `backend/schema.sql` (or `AI-Chat/api/schema.sql` if moved) in your Supabase SQL editor, then add `SUPABASE_URL` + `SUPABASE_SERVICE_KEY` to `AI-Chat/api/.env`. Works fine without it вҖ” falls back to in-memory storage.
